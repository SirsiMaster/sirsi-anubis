VS Code Extension
For any VS Code-based IDE
No AI Required
40 KB download

Works in VS Code, Cursor, Windsurf, Antigravity — anything built on the VS Code engine. Available on OpenVSX.

  • Guardian — Auto-renice LSPs to Background QoS (nice +10)
  • Memory GC — Restarts bloated language servers automatically
  • Status Bar — Live PANTHEON metrics, refreshed every 5s
  • Command Palette — Scan, ghost report, renice, optimal settings
  • Thoth — Context compression from .thoth/memory.yaml
🖥️
Menu Bar App
Always visible, IDE-independent
No AI Required
2.6 MB download

Native macOS menu bar application. Runs independently of any IDE — always visible, always monitoring. Auto-starts at login with LaunchAgent.

  • System tray icon — Heavyweight ankh, always visible
  • Live stats — RAM pressure, Git status, branch, accelerators
  • Command shortcuts — Quick-launch scan, guard, ghost hunt
  • Auto-start — LaunchAgent survives reboots, KeepAlive enabled
⌨️
CLI
Full power, scriptable, CI/CD ready
No AI Required
6.6 MB download

18 commands across 10+ modules. JSON output for piping. Works on macOS and Linux.

  • pantheon weigh — Full infrastructure scan (58 rules, 7 domains)
  • pantheon ka — Ghost detection (17 macOS locations)
  • pantheon guard — Process monitoring and resource control
  • pantheon maat — Quality governance and coverage
  • pantheon mirror — File deduplication with GUI
🤖
MCP Server
For AI-enabled IDEs
AI Required
Built-in

Model Context Protocol server lets AI coding assistants (Claude, Gemini, Copilot) query Pantheon directly — zero copy-paste.

  • scan_workspace — AI scans your system without you running commands
  • ghost_report — AI checks for dead processes
  • health_check — AI grades your system health
  • thoth_read_memory — AI reads project context
